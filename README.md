# CrazyLLM

## How to use it
Start [llm_server.py](llm_server.py) locally or on a remote system. If in a remote system, run [mount_llm_service.sh](mount_llm_service.sh) on your system to mount the temp folder to the one on the remote system (you might need to adjust paths). Then, start [llm_console.py](llm_console.py) on your local machine.
